# SnapURL - URL Shortener

SnapURL is a full-stack URL shortener built using FastAPI.
It allows users to shorten long URLs and access them easily through a clean UI.

---

## Features

* Shorten long URLs instantly
* Copy shortened link
* Redirect to original URL
* Error handling for invalid links
* Custom frontend UI

---

## Tech Stack

* FastAPI
* Python
* SQLite
* HTML, CSS, JavaScript

##Run Locally

```bash
pip install fastapi uvicorn sqlalchemy validators
uvicorn shortener_app.main:app --reload
```
